## Open Source Bookmark Manager (frontend)

Frontend for [Bookmarks](https://github.com/dignityinside/bookmarks).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
